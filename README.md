## **BTK AKADEMİ Python ile Makine Öğrenmesi Başlıklar**
Bu README dosyası, Python ile Makine Öğrenmesi kursunun amacını, içeriğini ve kullanımını açıklayan bir kaynaktır. Bu kurs, makine öğrenmesine giriş yapmak isteyen veya mevcut bilgisini derinleştirmek isteyen herkese yöneliktir. Kursun içeriği, temel makine öğrenmesi konularını ve bu alandaki popüler algoritmaları kapsar. Python programlama dili ve makine öğrenmesi için yaygın olarak kullanılan kütüphaneler, kurs boyunca kullanılan temel araçlardır. Bu repo, bu konuları kurs sırasında, kurs dışında ya da kurs sonrasında makine öğrenmesi ile ilgili temel bilgileri özetlemek için hazırlanmıştır.
